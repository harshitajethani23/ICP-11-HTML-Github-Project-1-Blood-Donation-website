Blood is classified into different groups based on the presence or absence of specific antigens. The main blood groups are:

1. Blood Type O

Contains no A or B antigens on red blood cells but has both anti-A and anti-B antibodies in plasma.

Can donate to: A, B, AB, O (Universal Donor)

Can receive from: O only

view more

2. Blood Type A

Contains A antigens on red blood cells and anti-B antibodies in plasma.

Can donate to: A, AB

Can receive from: A, O

view more

3. Blood Type B

Contains B antigens on red blood cells and anti-A antibodies in plasma.

Can donate to: B, AB

Can receive from: B, O

view more

4. Blood Type AB

Contains both A and B antigens on red blood cells and no antibodies in plasma.

Can donate to: AB only

Can receive from: A, B, AB, O (Universal Recipient)

view more

5. Rh Factor

Each blood type can be either Rh-positive (+) or Rh-negative (-), depending on the presence of the Rh antigen.
